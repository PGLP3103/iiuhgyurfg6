Como criar seu site?

Para começar crie uma pasta e coloque as imagens desejadas para seu site

abra o vs code e abra sua pasta

crie o index.html e o style.css

no index.html você fara toda parte escrita. Começe digitando html:5

use toda sua imaginaçao e crie seu site

use o style.css para definir varias coisas, como por exemplo: cor, tamanho, posição, largura, altura, etc.

no css você podera manipular suas imagens e definir todas as opçoes citadas acima

lembre-se sempre de criar um repositorio no Github ou no drive para salvar suas coisas

se você não salvar, provavelmente você perderá todo su progresso
